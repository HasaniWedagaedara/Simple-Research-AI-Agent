# Simple Research AI Agent Build Using Python

## Create a Python virtual environment


#### Remove previous venv

```bash
rm -r venv
```

#### create a vertual environment for windows

```bash
python -m venv venv 
source venv/Scripts/activate
```

#### install dependencies gloabally

```bash
pip install -r requirements.txt
```

## Run the System

```bash
streamlit app.py
```
